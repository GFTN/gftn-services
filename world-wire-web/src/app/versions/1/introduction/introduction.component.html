Introduction

API Reference

World Wire enables a network of geographically-dispersed payment providers to exchange value around the globe in seconds using blockchain-based digital assets for near real-time settlement.


Our API is based on widely understood RESTful concepts. All requests and responses are formatted in JSON as you'd expect.

Networks

World Wire has two different networks: Staging and Production. We use JWT for authentication, and both environments are hosted on separate architecture, meaning data does not flow between them. We've spent a lot of time making sure we get security right while keeping the onboarding experience nice and simple!


Staging
The first step of your journey is to jump into the Staging network. You'll be provisioned two nodes in order to simulate two Participants transacting on the World Wire network. When you're ready to get started, continue to the {{'Onboarding'}} section to contact us!

Production
Ladies and gentlemen, start your engines! Once you've proven your technology, we'll consolidate you onto a single node on the Staging network which will be a mirror of what we'll deploy into the Production network. The final steps involve a technology certification process conducted by our Onboarding team, to make sure everything's in order. You'll be transacting with real value in no time!

Support

If you encounter a problem or have a question, it's easy to contact World Wire support. Simply visit our Support portal and login with your IBM ID. If you're an existing client, you'll be able to open a case, which an agent will promptly respond to, otherwise you'll be prompted to create your IBM ID.
